# omp-skill-usage-tracker
This hook help you track the used skills, it will create a .csv at ~/.omp/skill-usage.csv that you can import in skill-usage-dashboard.html (after opening the .html in your browser) giving you a simple dashboard to see which skills are the most called

<img width="1852" height="981" alt="image" src="https://github.com/user-attachments/assets/74c6728d-7894-444b-8038-d2c832323e41" />

## Usage 
Just place the folder skill-usage-tracker in ~/.omp/agent/extensions 
then when you want to check you open the .html in your favourite browser and load the .csv
